# Release history

Index of every released version. The current release lives in
[`CHANGELOG.md`](../CHANGELOG.md) at the repository root; the files in
this folder group entries by major version.

| Line             | Coverage                                       | File                            |
| ---------------- | ---------------------------------------------- | ------------------------------- |
| **3.x** (current)| Active development — see the root              | [`../CHANGELOG.md`](../CHANGELOG.md) |
| 2.x              | v2.0.0, v2.1.0, v2.2.0                         | [`v2.x.md`](v2.x.md)            |
| 1.x              | v1.0.0, v1.0.1                                 | [`v1.x.md`](v1.x.md)            |
| Pre-fork         | Original `samp-rs` history (0.1.x – 0.9.x)     | [`historical.md`](historical.md)|

## Convention

- Each `vN.x.md` file collects every patch/minor release inside its
  major line.
- When a new major ships, the previous one is moved out of the root
  into the matching `vN.x.md`, and the root `CHANGELOG.md` is reset to
  the new release only.
- Release notes are written in English; commit messages stay in
  Portuguese, matching the project convention.
