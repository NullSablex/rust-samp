## [v1.0.0] — 2026/02/19

### Post-release fixes
- Renamed the project from "Rust-SAMP" to "rust-samp" in the README and LICENSE.
- Fixed doctests marked `compile_fail` that started compiling once the
  updated dependencies landed (`compile_fail` → `no_run`).
- Replaced `#[no_mangle]` with `#[unsafe(no_mangle)]` in the
  `samp-sdk` doctests (required by edition 2024).

### Modernization
- Bumped Rust edition from 2018 to **2024**.
- Added workspace `resolver = "3"`.
- Replaced `static mut` with `AtomicPtr` in `runtime.rs` and `encoding.rs`.
- Replaced `#[no_mangle]` with `#[unsafe(no_mangle)]` (required by edition 2024).
- Added explicit lifetimes where edition 2024 demands them (`amx.rs`).
- Replaced `std::mem::transmute` with `f32::to_bits().cast_signed()` in `repr.rs`.

### Dependencies
- `syn` 0.15 → 2.0, `proc-macro2` 0.4 → 1.0, `quote` 0.6 → 1.0.
- `bitflags` 1.0 → 2.10 (with the now-mandatory derives).
- `fern` 0.5 → 0.7.
- `memcache` 0.12 → 0.19.
- Removed the dead `colored` dependency (build-dep without a `build.rs`).

### Code clean-up
- `.map().flatten()` → `.filter_map()` in the codegen crate.
- `match / Err(_) => ()` → `if let` in `runtime.rs`.
- `unsafe { String::from_utf8_unchecked() }` → `String::from_utf8_lossy()` in `string.rs`.
- A shadowed `to_string()` (clashing with `Display`) was renamed to an internal `convert_to_string()`.
- Added precedence parentheses around a bit-shift expression.
- Renamed `Args::next()` to `Args::next_arg()` to avoid confusion with `Iterator`.
- Dropped unnecessary lifetimes (`strlen`, `add`).
- Added `# Safety` sections to `AmxPrimitive` and `AmxString::new`.

### Infrastructure
- Removed `.travis.yml`; added GitHub Actions (`.github/workflows/rust.yml`).
- Removed the obsolete `.rustfmt.toml` (`fn_args_density` was deprecated).
- Removed the legacy `docs/` folder (an old `cargo doc` output).
- Added an MIT `LICENSE` file at the repository root.
- Rewrote the README in Portuguese (later moved to English in v3.0.0).
- Added `CHANGELOG.md`.
- Removed stale branches (`potential-fix`, `pre-0.9.0`, `async-amx`).
